# Requirements
```bash
pip install -r requirements.txt
```

# Usage
```bash
git clone https://github.com/Moritz344/Twitch.Chat.git
cd Twitch-Chat
cd src
python3 main.py
```
Exit with Ctrl + c


# Info
you need to edit the secret.env and replace it with your ACCESS TOKEN.
